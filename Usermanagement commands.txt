Chage:
The chage command is used to view and change the user password expiry information. 
root@rsdmz:~/user_mgmt# chage -l shivani.gadekar

Last password change                                    : Jun 30, 2022
Password expires                                        : Sep 28, 2022
Password inactive                                       : never
Account expires                                         : never
Minimum number of days between password change          : 0
Maximum number of days between password change          : 90
Number of days of warning before password expires       : 2


chage -d 2018-12-01 root
use this option to set the last password change date to your specified date in the command

chage -E date username
Expiry

chage -m -M : Maximum and minimum

chage -W 2 username

Warning
chage -d 0 -M 90 -W 2
-M : Maximum days
-W : Warning days

finger
This command is used to procure information of the users on a Linux machine. You can use it on both local & remote machines

The syntax ‘finger’ gives data on all the logged users on the remote and local machine.

finger username.
